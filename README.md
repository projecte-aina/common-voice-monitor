> Fetch data from common voice and save it to a database for later analysis

## Prerequisites

- node >=16.14.2
- npm >=8.6.0
- pm2 >=5.2.0
- mongodb

## Install

```sh
npm install pm2 -g
```

```sh
npm install
```

## Usage

```sh
npm run start
```
